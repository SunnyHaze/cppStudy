# <center> ARRAY类模板，更安全的数组实现 <center>

## 这个模板的目的
为了解决传统数组模板容易越界的尴尬境地，实现了自动投出错误，并阻止对野地址进行进一步操作的数组。

## 这个模板现在可以实现什么
~~Array()~~ 
 
  默认构造函数已经被清空

    Array(int = 50)
通过此方式创建数组并声明其大小
    
    Array(const Array<T> &A);    
拷贝构造函数

    Array<T> &operator=(const Array<T> &rhs)

重载等于号，方便直接转移数组内变量，大小向下兼容，若试图将大数组赋给小数组，则报错退出。

    T &operator[](int);
重载下标运算符，实现对元素的快速访问，并在越界时提出警报并退出程序

    int ListSize() const;
返回数组当前最大长度。

    void Resize(int);
    重新设定数组最大长度

## 如何使用ARRAY
    直接向文件中添加头文件及源代码即可。
